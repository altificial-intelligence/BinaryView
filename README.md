## Usage
### Google Search
```
time python src/googlesearch.py saveUrls 'bernie sanders' 5 'face' 'y[1]' 'out/urls/berniesanders'
```
### Clarifai Classification
```
time python src/clarifai.py classify 'out/urls/' 'out/counts/'
```
