Learn More

Current image search algorithms utilize metadata, additional data about the image file itself. However, a novel technique in machine learning known as deep learning uses artificial intelligence to learn about and classify images. This open source program utilizes Google Custom Search to search for online images of the person's face and Clarifai to classify those images into lists of words, helping us to understand more about online personas.
